Getting Started

Installing
$ npm install @arkius/blockchain-lib

This is the npm library for Arkius Contracts. All the functions of the contracts can be accessed and run with this library.

You need to install the library first into your npm project with..

$ npm install @arkius/blockchain-lib

Now assuming youâ€™ve installed the blockchain-library, you need to create a file first, lets say test.js and import the library.

const arkius = require('@arkius/blockchain-lib') // Importing the arkius npm library

After importing the library, you need to setup your accounts, for which you need your accounts private key.

It's good to create a '.env' file to store all your accounts private key.

So create a .env file and write your private keys, it will look similar to this.

acc1_key = "YOUR_PRIVATE_KEY_ACCOUNT1"

Once your Environment file is setup you can import .env file into your main file that is test.js

require('dotenv').config({path:'PATH'}) // Importing the dotenv library

const arkius = require('@arkius/blockchain-lib') // Importing the arkius npm library

Structure of the Function

You can call the functions of the library, as described below:-

await arkius.module.METHOD(...parameter, index, gasReq)

Where:

  • await is used for async function.
  • arkius is an instance of the library.
  • module are the part of library. There is 1 module for every contract.
  • METHOD are the functions of the library in camel case.
  • ...parameter are the Parameters of functions.
  • index is the index of the Multiple logged in accounts.
  • gasReq is the amount of gas Required to execute the transaction.

Logging in Account

You can test your library by logging in account with your private key.

require('dotenv').config({path:'PATH'}) // Importing the dotenv library

const arkius = require('@arkius/blockchain-lib') // Importing the arkius npm library

await arkius.util.loginWithPrivateKey(process.env.acc1_key); // Loging in with private key

You can log in with metamask also.

const arkius = require('@arkius/blockchain-lib') // Importing the arkius npm library

await arkius.util.connectMetamask(); // Logging in with Metamask
