Arkius Governance Token

All the governance token related operations took place over here. But much of the operations are managed by token Manager such as minting and burning.

Structure

await arkius.token.Functions(...parameter, index, gasReq)

Where:

  • await is used for async function.
  • arkius is an instance of the library.
  • token is a module of library for Token Contract.
  • Functions are the functions of the Token Contract in camel case.
  • ...parameter are the Parameters of functions.
  • index is the index of the Multiple logged in accounts. Default is 0.
  • gasReq is the amount of gas Required to execute the transaction. Default value is already given.

(async) transfer(recepient, amount, index, gasReq)

This can be used to transfer locked and unlocked token to the recepient. Conditions:- 1) If the `recepient` is validated, they/it will receive locked tokens first from caller and then unlocked tokens. 2) If the caller is validated and sends:- i) locked tokens - `recepient` will receive locked tokens and will change to unlocked tokens. ii) unlocked tokens - `recepient` will receive locked tokens first from caller and then unlocked tokens. 3) If caller and `recepient` are not validated then they will transfer unlocked tokens only.

recepient address address of the recepient.

index uint 0 specify the account number index if you logged. in with multiple accounts, default is 0.

Transfers the unlocked token from `sender` to the `recepient`. Requirement - caller and `recepient` must not be validated.
