SeekerNFT

This is used for generating and burning Seeker NFT. Any member can generate their seekerNFT and become an Attention Seeker.

Structure

await arkius.roles.seeker.Functions(...parameter, index, gasReq)

Where:

  • await is used for async function.
  • arkius is an instance of the library.
  • roles is a module of library for Attention Seeker Contract.
  • seeker is a sub-module of roles library.
  • Functions are the functions of the Attention Seeker Contract in camel case.
  • ...parameter are the Parameters of functions.
  • index is the index of the Multiple logged in accounts. Default is 0.
  • gasReq is the amount of gas Required to execute the transaction. Default value is already given.
