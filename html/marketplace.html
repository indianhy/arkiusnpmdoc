Marketplace

These functions can only be called by contract owner. Contract owner can serve the ads, or can calls a function once the ad got clicked or expired.

Structure

await arkius.marketplace.Functions(...parameter, index, gasReq)

Where:

  • await is used for async function.
  • arkius is an instance of the library.
  • marketplace is a module of library for Marketplace Contract.
  • Functions are the functions of the Marketplace Contract in camel case.
  • ...parameter are the Parameters of functions.
  • index is the index of the Multiple logged in accounts. Default is 0.
  • gasReq is the amount of gas Required to execute the transaction. Default value is already given.
