Token Manager

This is used for managing the governance tokens. Minting and Burning of the tokens can be managed by token manager only. And member can buy Arkius Tokens in exchange of AVAX with 'buy' function.

Structure

await arkius.manager.Functions(...parameter, index, gasReq)

Where:

  • await is used for async function.
  • arkius is an instance of the library.
  • manager is a module of library for TokenManager Contract.
  • Functions are the functions of the TokenManager Contract in camel case.
  • ...parameter are the Parameters of functions.
  • index is the index of the Multiple logged in accounts. Default is 0.
  • gasReq is the amount of gas Required to execute the transaction. Default value is already given.

(async) buy(minReceived, amount, index, gasReq)

This function is used to buy buy Arkius tokens with Avax. Member can send Avax and can receive Tokens in exchange. Note:- If there is sudden increase in price of Arkius Token, and due to this inflation the member will be getting less tokens than `minReceived`, in this condition the function will revert else the member will receive tokens as per the Avax provided.
